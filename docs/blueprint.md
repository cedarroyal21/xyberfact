# **App Name**: FactLens

## Core Features:

- URL Capture: Captures the URL of the current webpage in the browser.
- Context Analysis: Analyzes the text content of the webpage.
- Fact Verification Tool: Uses a tool to assess the factuality of claims and identify potential biases or inaccuracies in the text.
- Source Evaluation: Evaluates the reliability and credibility of the source website based on domain authority, reputation, and transparency.
- Highlight Discrepancies: Highlights potential discrepancies, biases, or inaccuracies directly on the webpage.
- Trust Score: Generates an overall trust score for the information presented on the webpage.

## Style Guidelines:

- Primary color: Deep blue (#2E3192) to convey trust and reliability.
- Background color: Light gray (#F0F2F5) for a clean and unobtrusive interface.
- Accent color: Soft green (#8AC53E) to indicate verified or reliable information.
- Body and headline font: 'Inter', a sans-serif font, for clarity and readability.
- Use clear and recognizable icons to represent different levels of trust and verification status.
- Display key information (trust score, source evaluation) prominently in a concise manner.
- Subtle animations when highlighting discrepancies.
